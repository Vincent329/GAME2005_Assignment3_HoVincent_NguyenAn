Instructions:

Click on left or right buttons to go to either Scene 1 or Scene 2 respectively

Scene 1:
WASD to move
Adjust variables with IMGUI
Click Play Scene to start spawning bullets

Scene 2:
Simulation is paused at the start
Adjust variables prior to activating the scene
Press Backtick to deactivate ImGUI and move the paddle with the mouse
Press Backtick again to pause the screen

For either scene:
Click Back to go to the start
Click Next to go to the end scen

End Scene:
same functionality as start screen
